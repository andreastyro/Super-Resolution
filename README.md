# Super Resolution
This project explores and compares multiple deep learning approaches to Single Image Super Resolution using the DIV2K dataset, developed as part of the Applied Machine Learning Systems II coursework.

The repository includes two models:

- SRCNN, a baseline convolutional neural network for image upscaling
- SRGAN, a generative adversarial network designed to produce high-resolution images with enhanced perceptual quality

The goal of this project is to evaluate the trade-offs between reconstruction accuracy and perceptual realism, highlighting the strengths and limitations of each approach.
